# Blender-SRT-Loader [RU | [EN](./README_EN.md)]
  Независимый аддон для Blender, реализующий поддержку импорта файлов формата .srt. Данный формат часть технологии SpeedTree® и встречается в играх на базе движка BigWorld. Аддон создан исключительно для обеспечения совместимости и не аффилирован с IDV или иными правообладателями.

## Возможности
- Импорт геометрии из `.srt` версии 06.0.0.
- Автоматическое создание материалов с основными текстурами (`.dds`).
- Группировка по уровням детализации (LOD) в коллекции.
- Корректные UV и кастомные нормали.

## Установка
1. Скачайте ZIP-архив репозитория.
2. В Blender: `Edit → Preferences → Add-ons → Install…`, выберите архив.
3. Включите аддон «Import-Export: SRT Loader (.srt)».

## Использование
- Меню `File → Import → SpeedTree SRT (.srt)`.
- Выберите `.srt` файл.
- После импорта в сцене появятся коллекции: `<имя>_SRT` и вложенные `<имя>_LOD0`, `<имя>_LOD1`…
- Текстуры ищутся в папке с импортированным файлом.
